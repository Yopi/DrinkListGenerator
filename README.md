dgenerate
========
**dgenerate** (dɪˈdʒɛn(ə)rət/) is a drink list generator

Installation:
```
$ gem install bundler
$ bundle

```

Usage:
```
$ rake beer
or
$ rake drinks
```

* Print the resulting html page as a pdf.
* For drinks you need to zoom out and print screen the page and then make it a pdf.
* Print the pdf.
* Done!
